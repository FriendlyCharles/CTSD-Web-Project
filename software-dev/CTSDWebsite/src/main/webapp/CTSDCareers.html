Bachelor of Applied Science in Computing Technology and Software Development

Career Outlook

The Bachelor of Applied Science in Computing Technology and Software Development prepares students for careers or graduate studies. The program teaches students how to write and debug software code and how to structure, query and integrate databases. Students learn about each aspect of the software development lifecycle (SDLC): planning, designing, implementing, testing, deploying, and maintaining software solutions. In addition, students learn project management skills and how to communicate technical concepts orally and in writing. The curriculum prepares students to solve practical, real-world problems.

Potential Career Paths

  • Full-Stack Software Developers
  • UI/UX Developers
  • System Software Developers
  • Computer and Information Systems Managers
  • Computer Systems Analysts

    • The Bureau of Labor Statistics (BLS) projects employment of software developers, quality assurance analysts, and testers to grow 25 percent from 2021 to 2031, well above the average for all occupations of five percent.
