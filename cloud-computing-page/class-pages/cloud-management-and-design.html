CLOUD MANAGEMENT AND DESIGN Prerequisite: Admission to the B.A.S. Degree in Computing Technology and Software Development Students will learn how to manage and design cloud services. Students will learn how to create virtual cloud machines, design storage architecture, and build appropriate processes. Students will learn how security and design are crucial to move platforms in and out of the cloud. Case studies will be used to showcase the application of these services to the modern cloud platform. Student will design services and redundancy in a global infrastructure
